# Spotify Connection & Settings Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Settings page with Spotify connection, vault info, and theme preferences, plus a contextual Spotify connection prompt in the scene's Add Track dialog.

**Architecture:** A shared OAuth utility (`spotify-auth.ts`) handles all Spotify auth logic. The Settings page (`/settings`) uses shadcn-svelte Tabs for three sections. The scene dialog's Spotify tab checks auth status and shows an inline connection card when disconnected. The sidebar footer gets a gear icon linking to Settings, and the theme toggle moves to the Preferences tab.

**Tech Stack:** Svelte 5 (runes), SvelteKit, shadcn-svelte (Tabs component), Tauri IPC (`invoke`/`listen`), mode-watcher, svelte-sonner, Lucide icons

### Task 2: Create shared Spotify auth utility

**Files:**
- Create: `src/lib/utils/spotify-auth.ts`

- [ ] **Step 1: Create the utility file**

Create `src/lib/utils/spotify-auth.ts`:

```ts
import { invoke } from "@tauri-apps/api/core";
import { listen } from "@tauri-apps/api/event";
import { openUrl } from "@tauri-apps/plugin-opener";
import type { SpotifyAuthStatus } from "$lib/types/vault";

export async function getSpotifyStatus(): Promise<SpotifyAuthStatus | null> {
  return invoke<SpotifyAuthStatus | null>("spotify_get_auth_status");
}

export async function connectSpotify(): Promise<SpotifyAuthStatus> {
  const authUrl = await invoke<string>("spotify_start_auth_flow");

  // Set up listener BEFORE opening the URL so we don't miss the callback
  const statusPromise = new Promise<SpotifyAuthStatus>((resolve, reject) => {
    listen<{ code: string; state: string }>("spotify-auth-callback", async (event) => {
      try {
        const result = await invoke<SpotifyAuthStatus>("spotify_exchange_code", {
          code: event.payload.code,
          state: event.payload.state,
        });
        resolve(result);
      } catch (e) {
        reject(e);
      }
    });
  });

  // Now open the browser for the user to authorize
  await openUrl(authUrl);

  return statusPromise;
}

export async function disconnectSpotify(): Promise<void> {
  await invoke("spotify_revoke");
}
```
