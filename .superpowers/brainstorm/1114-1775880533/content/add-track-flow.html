Adding Tracks — Interaction Model

How should the GM add new audio to a scene? The backend supports local files (copied into vault) and Spotify URIs.

A

Simple File Picker Only (v1)

Click "+ Add Track" → system file dialog opens → pick an audio file → it's copied to the vault and added as a slot. Spotify support deferred to later. Fastest to build, covers the core use case.

Supports: .mp3, .wav, .ogg, .flac
B

Dialog with Source Tabs

Click "+ Add Track" → a dialog opens with two tabs: Local File (file picker button) and Spotify (paste a track/playlist/album URI). Handles both sources cleanly in one place.
